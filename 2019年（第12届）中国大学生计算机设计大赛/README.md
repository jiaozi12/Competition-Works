### 项目简述
基于tello无人机的图像目标检测与路径规划算法

### 设计概要
1.无人机飞控和图像传输模块：用于控制无人机飞行距离、转向角度等行动，传输无人机摄像头拍摄到的实时视频流。
2.图像目标检测模块：用于识别无人机视角中的水面垃圾，并获取各处水面垃圾相对于无人机的坐标信息。
3.路径规划模块：根据图像目标检测模块得到的垃圾坐标进行路径规划，无人机再根据规划好的路线依次完成作业。
